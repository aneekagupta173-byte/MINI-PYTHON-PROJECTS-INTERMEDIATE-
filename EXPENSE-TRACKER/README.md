
# 💰 Expense Tracker

A modern and user-friendly **Expense Tracker** built with **Python** and **Tkinter** that helps users manage their daily expenses efficiently. The application allows users to record, view, and delete expenses while providing a **cumulative spending trend graph** to visualize spending habits over time.

---

## ✨ Features

- ➕ Add new expenses with details
- 📋 View all recorded expenses
- 🗑️ Delete existing expenses
- 💰 Automatically calculate total expenses
- 📈 Interactive cumulative spending trend graph
- 💾 Save expense records locally
- 🖥️ Clean and intuitive graphical user interface
- ⚡ Fast and lightweight application

---

## 🛠️ Tech Stack

- **Language:** Python
- **GUI Framework:** Tkinter

- **Visualization:** Matplotlib

## 🚀 Getting Started

### Prerequisites

- Python 3.8 or above

### Installation

Clone this repository:

```bash
git clone https://github.com/your-username/expense-tracker.git
```

Navigate to the project folder:

```bash
cd expense-tracker
```

Install the required libraries:

```bash
pip install matplotlib
```

Run the application:

```bash
python expense_tracker.py
```

---

## 📖 How to Use

1. Launch the application.
2. Enter the expense details.
3. Click **Add Expense** to save the record.
4. View all saved expenses.
5. Delete any unwanted expense.
6. Monitor the total amount spent.
7. View the **Cumulative Spending Trend Graph** to analyze your spending habits over time.

---

## 📊 Cumulative Spending Trend

One of the key features of this project is the **Cumulative Spending Trend Graph**.

The graph automatically updates whenever a new expense is added and allows users to:

- Track total spending over time
- Identify periods of high expenditure
- Understand spending patterns
- Improve budgeting decisions

This visualization makes expense tracking more meaningful by converting raw financial data into an easy-to-understand graphical representation.
